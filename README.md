# RoadSOS
RESTful emergency response backend built with Java 17 &amp; Spring Boot 3.5 — locates nearest hospitals, ambulances &amp; trauma centres using Haversine algorithm with JWT-secured APIs

#ReadMe
Road SOS is a location-aware emergency response API that connects accident 
victims to the nearest trauma centres, ambulances, and police units across 
Tamil Nadu — built with Spring Boot 3.5, JWT Security, and offline-resilient 
architecture.
